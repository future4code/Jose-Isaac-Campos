RGB generator

O padrão rgb é compostos por três cores: vermelho, verde e azul. Cada cor pode assumir valores de 0 á 255, a veriação desse valores é o que compoe as cores na web.

Nesse projeto o usuário pode ir movimentando os valores para cada cor e ver em tempo real qual cor está sendo formada como támbém o valor rgb para a mesma
